时时彩技巧：复式追号玩法攻略

  　　时时彩在时时彩的彩种中是比较有优势的一个彩种，因为其奖金比其他时时彩玩法的奖金高，这样在选号上就更加的有优势。我们来看看几个有优势的玩法：
 
　 　三星直选：时时彩三星直选的玩法是竞猜开奖号码的后面三位，选择三个或者三个以上号码进行投注，投注号码和开奖号码的后面三位相同则中奖，单注奖金 1160元。看清楚咯，是1160元，而不是其他时时彩彩种玩法的1000元。大家都知道，三星直选玩法的中奖率并不高，所以我们采用的方式大多为追号或 者复式的方式来玩此玩法，追号和复式的方式投注靠的就是资金，因为奖金比较高，所以我们追号的成功率也就相对较大一点。复式是同样的道理。
 
　　四星直选：四星直选在时时彩里面是一个比较特殊的玩法，因为时时彩四星直选玩法的一等奖的奖金是10000元整，且此玩法中奖不上税，所以更加的利于做追号的计划，复式同理，这样中奖的几率和三星一样，也是更加的大一些。
 
　　五星的玩法的奖金是116000元，高于其他玩法的100000元。其他小盘玩法的奖金也明显的高于其他省市的时时彩彩种的奖金。小盘玩法主要集中在复式的方式上面。所以高奖金使我们有更多的选择去投注，从而有更高的中奖的几率。
